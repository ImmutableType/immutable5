# Changelog

All notable changes to ImmutableType will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2026-01-06

### Added
- Flow EVM Wallet support via EIP-6963
- Unified wallet system supporting MetaMask and Flow Wallet
- WalletSelector component for multi-wallet selection
- EIP-6963 wallet discovery implementation
- Automatic profile creation redirect for new users
- Cadence wallet detection with clear error messages
- Comprehensive technical documentation

### Changed
- Updated all services to accept generic BrowserProvider
- Refactored wallet connection to use unified system
- Updated Navigation to use unified wallet hook
- Improved error messages for wallet detection issues

### Fixed
- TypeScript errors in event listeners
- Navigation not detecting wallet connection state changes
- WalletSelector not showing Flow Wallet option
- Profile creation page not auto-selecting wallet auth
- Build errors with archived code

## [0.1.0] - 2025-12-XX

### Added
- Initial release
- Profile NFT system
- Bookmark NFT system
- MetaMask wallet integration
- Token qualification system
- BUFFAFLOW integration
